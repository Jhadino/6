@startuml
title Процесс взаимодействий в киберспортивной команде

actor Игрок
actor ТренерскийШтаб
actor Аналитик
actor Менеджер
actor ТехническийПерсонал
actor Фанаты

participant "Киберспортивная Команда" as Team
database "База Данных" as Database
participant "Стратегия" as Strategy
participant "Фан База" as FanBase
participant "Турнир" as Tournament

== Сценарий 1: Подготовка к турниру ==
Менеджер -> Team: Запрос информации о турнире
Team -> Tournament: Узнать детали турнира
Tournament --> Team: Данные о турнире
Team -> Менеджер: Уведомление о турнире
Менеджер -> Team: Подготовка планов
Team -> ТренерскийШтаб: Создание расписания тренировок
ТренерскийШтаб -> Игрок: Уведомление о тренировках
ТренерскийШтаб -> Аналитик: Запрос на анализ противников
Аналитик -> Strategy: Разработка стратегии
Strategy --> Team: Утверждение стратегии
ТехническийПерсонал -> Team: Настройка оборудования для турнира

== Сценарий 2: Игровой анализ после матча ==
Игрок -> Team: После матча - анализ выступления
Team -> Аналитик: Запрос на анализ игры
Аналитик -> Team: Подготовка отчета о матче
Team --> ТренерскийШтаб: Отчет о результатах
ТренерскийШтаб -> Игрок: Обсуждение анализа
ТренерскийШтаб -> ТехническийПерсонал: Запрос на проверку оборудования
Аналитик -> Database: Сохранение статистики матча

== Сценарий 3: Дисциплинарные меры ==
ТренерскийШтаб -> Team: Уведомление о дисциплинарном нарушении
Team -> Игрок: Уведомление о нарушении
Игрок -> Team: Объяснение ситуации
Team -> ТренерскийШтаб: Обратная связь от игрока
ТренерскийШтаб -> Database: Запись дисциплинарных мер
Менеджер -> Team: Уведомление о последствиях

== Сценарий 4: Организация общения с фанатами ==
Менеджер -> Team: Инициирование общения с фанатами
Team -> Фанаты: Обратная связь через соцсети
Фанаты -> Team: Вопросы и комментарии
Team -> Игрок: Подготовка ответов к фанатам
Игрок -> FanBase: Ответы на вопросы фанов
Менеджер -> FanBase: Публикация ответов

== Сценарий 5: Оценка производительности команды ==
ТренерскийШтаб -> Team: Запрос на оценку производительности
Team -> Аналитик: Сбор статистики по играм
Аналитик -> Database: Запрос данных о матчах
Database --> Аналитик: Данные о производительности
Аналитик -> Team: Подготовка отчета
Team --> ТренерскийШтаб: Презентация результатов
Менеджер -> Team: Определение дальнейших шагов

@enduml
